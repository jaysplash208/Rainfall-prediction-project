 Rainfall Prediction Model
Predicting precipitation using Machine Learning Classification.*

# The Goal
Built a model to predict whether it will rain tomorrow based on current meteorological data. This demonstrates my ability to handle real-world datasets and implement machine learning pipelines.

# Tech 
Tools:*Python, Google Colab
Libraries:Scikit-learn, Pandas
Models: Logistic Regression and Decision Trees

# Key Insight
Humidity and atmospheric pressure were the strongest indicators of rainfall in this dataset.
